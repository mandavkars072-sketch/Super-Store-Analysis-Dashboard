# 📊 Sales Analysis Dashboard using Microsoft Excel

## 🔎 Project Overview

This project focuses on analyzing retail sales data and building an interactive Excel dashboard to generate actionable business insights. 

The dashboard provides a clear view of sales performance, profitability, regional trends, and customer contribution using advanced Excel features.

## 🎯 Business Problem

Businesses need a centralized and interactive view of their sales performance to:
- Track revenue and profit trends
- Identify top-performing regions and categories
- Analyze customer contribution
- Support data-driven decision making
  
## 🛠 Tools & Skills Used

- Microsoft Excel
- Data Cleaning & Data Preprocessing
- Pivot Tables & Pivot Charts
- Slicers & Interactive Filtering
- KPI Cards
- Conditional Formatting
- Data Visualization
- Business Insight Generation

## 📈 Dashboard Features

- Total Sales, Profit & Order KPIs
- Region-wise Sales Analysis
- Category-wise Profit Analysis
- Monthly Sales Trend
- Top Customers by Revenue
- Interactive Filters using Slicers

## 🔍 Key Insights

- West region generated the highest overall revenue.
- Technology category delivered maximum profit margins.
- Sales peaked during Q4, indicating seasonal demand.
- Top 20% customers contributed a major portion of revenue.
